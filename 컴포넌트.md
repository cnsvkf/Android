### 2. Service

- mvvm 속 service 개념과 다름

- Activity은 화면이 있음 / Service은 화면 없음

- 백그라운드에서 작업

    -> onStop은 대기 상태, Sevice은 작업 수행

    ->___예시___ : 유튜브 뮤직에서 onstop 일 시 멈추지만, service이기에 재생됨



#### 2.5 startService() / bindService() 

![alt text](image-1.png)

- startService() : Activity와 연결 안됨 -> Activity가 죽어도 계속 실행( 독립 실행 )

- bindService() : Activity와 연결 -> 데이터 주고받을 수 있음 -> Activity 죽으면 끝
