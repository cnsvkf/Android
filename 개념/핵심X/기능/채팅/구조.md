1. 사용자가 입력창에 메시지를 입력
2. 전송 버튼 클릭
3. ViewModel이 빈 문자열인지 검사
4. Repository가 Firestore에 메시지 저장
5. Firestore에 붙어 있던 실시간 리스너가 새 메시지 변화를 감지

6. 메시지 목록이 다시 내려옴
7. ViewModel의 StateFlow가 갱신됨
8. Compose UI가 자동으로 다시 그려짐
