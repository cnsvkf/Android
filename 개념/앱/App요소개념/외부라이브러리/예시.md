## 도서관 라이브러리 

    화면 진입 / 검색
            ↓
    ViewModel에서 검색 함수 호출
            ↓
    Repository
            ↓
    Retrofit Service
            ↓
    Read365/DLS API
            ↓
    책 제목, 저자, ISBN, 표지 이미지 URL 응답
            ↓
    ViewModel이 UiState 갱신
            ↓
    Compose 화면이 자동 갱신
            ↓
    Coil AsyncImage가 coverUrl로 이미지 로딩

| 라이브러리      | 역할            | 지금 앱에서 쓰는 곳                             |
| ---------- | ------------- | --------------------------------------- |
| `Retrofit` | 외부 API 호출     | Read365/DLS에서 책 목록, 책 상세 정보 받아오기        |
| `OkHttp`   | 네트워크 요청 공통 처리 | 로그, 헤더, 인증 토큰, 공통 파라미터 처리               |
| `Coil`     | 이미지 URL 로딩    | 책 표지 이미지 보여주기                           |
| `Hilt`     | 의존성 주입        | Repository, ApiService, OkHttpClient 관리 |
