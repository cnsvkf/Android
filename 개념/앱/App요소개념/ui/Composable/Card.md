```kotlin
Card {
    Text("프로젝트 제목")
}
```
| 역할     | 설명                           |
| ------ | ---------------------------- |
| UI 묶음  | 제목, 설명, 이미지 등을 하나의 카드처럼 묶음   |
| 배경색 적용 | `containerColor`로 카드 배경색 지정  |
| 그림자 적용 | `elevation`으로 카드가 떠 있는 느낌 지정 |
| 클릭 처리  | `clickable`을 붙여 카드 클릭 가능     |
| 둥근 모서리 | 기본적으로 Material 디자인 스타일 적용    |
