- #### drawCircle(), drawArc() 같은 그리기 메서드를 사용할 수 있게 해주는 범위

    - Canvas, drawBehind, drawWithContent, drawWithCache 등에서 사용



| 상황                | 사용                |
| ----------------- | ----------------- |
| 그림만으로 된 UI를 만들 때  | `Canvas`          |
| 기존 UI 뒤에 그림을 넣을 때 | `drawBehind`      |
| 기존 UI 앞뒤 모두 제어할 때 | `drawWithContent` |
| 계산 결과를 캐싱해야 할 때   | `drawWithCache`   |
