| 종류            | 대표 API                                                           | 언제 씀                       |
| ------------- | ---------------------------------------------------------------- | -------------------------- |
| 나타남 / 사라짐     | `AnimatedVisibility`                                             | 카드, 버튼, 안내 문구 보이기/숨기기      |
| 크기 변화         | `animateContentSize()`                                           | 접기/펼치기 카드                  |
| 내용 전환         | `AnimatedContent`                                                | 숫자, 상태, 화면 내용 변경           |
| 부드러운 페이드 전환   | `Crossfade`                                                      | A 화면 ↔ B 화면 전환             |
| 단일 값 애니메이션    | `animateFloatAsState`, `animateDpAsState`, `animateColorAsState` | 투명도, 크기, 위치, 색상            |
| 여러 값 동시 애니메이션 | `updateTransition` / `Transition`                                | 선택 상태에 따라 크기+색+투명도 같이 변경   |
| 무한 반복         | `rememberInfiniteTransition`                                     | 로딩 점, 반짝임, skeleton 효과     |
| 직접 제어         | `Animatable`                                                     | 드래그, 스와이프, 터치 기반 이동        |
| 리스트 아이템 변화    | `animateItem()`                                                  | LazyColumn 아이템 추가/삭제/위치 변경 |
| 화면 간 연결 애니메이션 | shared element transition                                        | 리스트 이미지 → 상세 이미지 연결        |
| 벡터 이미지 애니메이션  | `AnimatedVectorDrawable`                                         | 아이콘 모양 변화, 마이크로 애니메이션      |



## “어떤 애니메이션 API를 골라야 하는지” 의 결정 트리

     "이미지/SVG 기반" 이면 AnimatedVectorDrawable이나 Lottie 같은 프레임워크를 고려
     
     "무한 반복" 이면 rememberInfiniteTransition

     "화면 내용 전환" 이면 AnimatedContent/Crossfade

     "보이기/숨기기" 면 AnimatedVisibility

     "크기 변화" 면 animateContentSize

     "독립적인 값 하나" 면 animate*AsState 

     "제스처 기반" 이면 Animatable을 고른다.


## 애니메이션 효과


| 함수                       | 의미           | 사용 상황          |
| ------------------------ | ------------ | -------------- |
| `fadeIn()`               | 서서히 나타남      | 로딩, 메시지, 화면 전환 |
| `fadeOut()`              | 서서히 사라짐      | 기존 UI 제거       |
| `slideIn()`              | 밀려 들어옴       | 카드, 화면, 바텀시트   |
| `slideOut()`             | 밀려 나감        | 뒤로가기, 탭 전환     |
| `slideInHorizontally()`  | 가로 방향으로 들어옴  | 좌우 화면 전환       |
| `slideOutHorizontally()` | 가로 방향으로 나감   | 상세 화면 이동       |
| `slideInVertically()`    | 세로 방향으로 들어옴  | 에러 메시지, 알림     |
| `slideOutVertically()`   | 세로 방향으로 나감   | 상단 메시지 제거      |
| `scaleIn()`              | 작게 시작해서 커짐   | 팝업, 버튼 강조      |
| `scaleOut()`             | 작아지며 사라짐     | 팝업 닫기          |
| `expandIn()`             | 크기가 확장되며 나타남 | 접힌 영역 열기       |
| `shrinkOut()`            | 크기가 줄며 사라짐   | 접힌 영역 닫기       |
