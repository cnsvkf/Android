| 종류             | 용도              | 대표 API             |
| -------------- | --------------- | ------------------ |
| 경고·확인 창        | 삭제 확인, 로그아웃 확인  | `AlertDialog`      |
| 자유로운 커스텀 창     | 이미지, 입력창, 여러 버튼 | `Dialog`           |
| 특정 컴포넌트 주변에 표시 | 말풍선, 작은 선택 창    | `Popup`            |
| 항목 선택 메뉴       | 더보기 메뉴, 필터 선택   | `DropdownMenu`     |
| 화면 아래에서 올라오는 창 | 필터, 옵션 선택       | `ModalBottomSheet` |
| 잠시 안내하는 메시지    | 저장 완료, 실패 안내    | `Snackbar`         |


## Dialog 종류

| 종류                    |   제공되는 기본 UI |   자유도 | 대표 사용 상황                      |
| --------------------- | -----------: | ----: | ----------------------------- |
| `AlertDialog`         |           많음 |    낮음 | 삭제 확인, 로그아웃 확인, 오류 안내         |
| `BasicAlertDialog`    |           일부 | 중간~높음 | Material 스타일의 커스텀 팝업          |
| `Dialog`              |        거의 없음 | 가장 높음 | 이미지, 폼, 복잡한 버튼, 커스텀 크기        |
| `DatePickerDialog`    |      날짜 선택 틀 |    중간 | 생년월일, 예약 날짜 선택                |
| 시간 선택 Dialog          |        직접 조합 |    높음 | 예약 시간, 알람 시간 선택               |
| Navigation `dialog()` | 화면 이동과 상태 관리 | 별도 개념 | Dialog 자체를 Navigation 목적지로 관리 |
