> ## 기본 개념

앱을 구조하는 방법( 구조 설계 )

---

- View 만들기

- ViewModel 만들기

- Repository 만들기


은 아키텍처 구현 작업

-> 아키텍처로 View 등을 설계하고 View, ViewModel 등으로 구현한 후 배치하는 과정


-> ___예시___

```
버튼을 누르면 숫자가 1 증가하는 앱 

[View]
화면에 숫자 보여주기
버튼 클릭 받기

   ↓

[ViewModel]
버튼 클릭 요청 받기
숫자 증가 처리

   ↓

[Repository]
현재 숫자 저장/반환 // 서버 데이터가 필요 없기에 레포에서 끝
```
